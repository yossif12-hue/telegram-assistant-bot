# بوت تيليجرام للموظفين والطلاب - مع مساعد ذكي (AI Chat)

هذا البوت مصمم ليكون مساعداً شخصياً فعالاً للموظفين والطلاب في مهامهم اليومية. يوفر البوت أدوات لإدارة المهام، تقديم نصائح دراسية، والآن يتضمن مساعداً ذكياً يعتمد على الذكاء الاصطناعي للإجابة على استفساراتك وتقديم الدعم في مجموعة واسعة من المواضيع.

## الميزات الرئيسية
- **إدارة المهام**: إضافة، عرض، وحذف المهام اليومية.
- **دعم الطلاب**: تقديم نصائح دراسية عشوائية ومفيدة.
- **مساعد ذكي (AI Chat)**: يمكنك الدردشة مع البوت وطرح الأسئلة عليه للحصول على إجابات ومساعدة فورية، تماماً مثل ChatGPT.
- **سهولة الاستخدام**: واجهة بسيطة تعتمد على الأوامر النصية والدردشة الحرة.
- **كفاءة عالية**: استجابة سريعة وتصميم موثوق.

## كيفية التشغيل
1. تأكد من تثبيت المكتبات اللازمة:
   ```bash
   pip install python-telegram-bot openai
   ```
2. قم بتشغيل ملف البايثون:
   ```bash
   python main.py
   ```

## قائمة الأوامر المتاحة
| الأمر | الوصف |
| :--- | :--- |
| `/start` | بدء المحادثة مع البوت. |
| `/help` | عرض قائمة المساعدة والأوامر. |
| `/addtask <مهمة>` | إضافة مهمة جديدة لقائمتك. |
| `/tasks` | عرض جميع المهام الحالية. |
| `/donetask <رقم>` | وضع علامة "مكتملة" على مهمة معينة. |
| `/studytips` | الحصول على نصيحة دراسية مفيدة. |
| **الدردشة الحرة** | أرسل أي رسالة للبوت وسيقوم المساعد الذكي بالرد عليك. |

## ملاحظات أمنية
يحتوي ملف `main.py` على التوكن الخاص بك. يرجى التأكد من عدم مشاركة هذا الملف علناً في المستودعات العامة (مثل GitHub) دون استخدام متغيرات البيئة لحماية التوكن.

**ملاحظة:** يتم استخدام نموذج `gpt-4o` افتراضياً للمساعد الذكي. يمكنك تغيير هذا النموذج في الكود إذا كنت تفضل نموذجاً آخر متاحاً عبر OpenAI API.
