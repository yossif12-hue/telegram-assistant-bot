# بوت تيليجرام للموظفين والطلاب

هذا البوت مصمم ليكون مساعداً شخصياً فعالاً للموظفين والطلاب في مهامهم اليومية. يوفر البوت أدوات لإدارة المهام، تقديم نصائح دراسية، وتسهيل التواصل.

## الميزات الرئيسية
- **إدارة المهام**: إضافة، عرض، وحذف المهام اليومية.
- **دعم الطلاب**: تقديم نصائح دراسية عشوائية ومفيدة.
- **سهولة الاستخدام**: واجهة بسيطة تعتمد على الأوامر النصية.
- **كفاءة عالية**: استجابة سريعة وتصميم موثوق.

## كيفية التشغيل
1. تأكد من تثبيت مكتبة `python-telegram-bot`:
   ```bash
   pip install python-telegram-bot
   ```
2. قم بتشغيل ملف البايثون:
   ```bash
   python main.py
   ```

## قائمة الأوامر المتاحة
| الأمر | الوصف |
| :--- | :--- |
| `/start` | بدء المحادثة مع البوت. |
| `/help` | عرض قائمة المساعدة والأوامر. |
| `/addtask <مهمة>` | إضافة مهمة جديدة لقائمتك. |
| `/tasks` | عرض جميع المهام الحالية. |
| `/donetask <رقم>` | وضع علامة "مكتملة" على مهمة معينة. |
| `/studytips` | الحصول على نصيحة دراسية مفيدة. |

## ملاحظات أمنية
يحتوي ملف `main.py` على التوكن الخاص بك. يرجى التأكد من عدم مشاركة هذا الملف علناً في المستودعات العامة (مثل GitHub) دون استخدام متغيرات البيئة لحماية التوكن.
